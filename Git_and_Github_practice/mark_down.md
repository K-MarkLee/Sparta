# 마크다운으로 글 적어보기 실험

## 마크다운이란?

마크 다운은 가볍고 간단한 문서 형식 언어로, 텍스트를 쉽게 읽고 쓰기 위해 만들어졌다.  블로그 같은 경우도 우측 상단에 보면, markdown으로 글을 적을 수 있게 변경 할 수 있다.

```markdown
이것은 마크다운의 예시 입니다
# 이것은 H1 제목
## 이것은 H2 제목
### 이것은 H3 제목
```
